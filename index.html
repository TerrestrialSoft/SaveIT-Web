How does it work?

Application functionality

Our application revolutionizes multiplayer gaming by allowing effortless sharing of game progress among users in supported games. Say goodbye to the hassle of waiting for someone to start a server with your shared game save. With our solution, you can seamlessly continue your adventure with friends anytime, anywhere.

When a user wants to start playing a game, they simply launch the SaveIt application, select a previously created profile, and press the play button. All game progress is versioned and stored in the cloud. Before the game starts, the latest game progress is downloaded from the cloud storage, and the storage is locked to prevent conflicts caused by multiple versions running simultaneously. The game then launches on the player’s device.
After finishing the game, the user needs to press the Update and Close button, which synchronizes the game progress back to the cloud storage and unlocks the storage so that other users can download the updated game progress.
